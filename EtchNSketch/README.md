# Overview

A minimal Turtle-based Etch-n-Sketch: a Turtle controlled by keyboard input to draw on the screen. Simple movement, pen control, and a screen-clear command are provided.

### Keybindings

- Arrow Up — move forward (10 units)

- Arrow Down — move backward (10 units)

- Arrow Left — turn left (10°)

- Arrow Right — turn right (10°)

- Space — clear / reset the turtle and drawing

- Left Shift (Shift_L) — lift pen (stop drawing)

- z — lower pen (resume drawing)


### Requirements

Python (3.x) with the built-in turtle module.

### Run

Save the script (e.g., main.py) and run:

'''python

python main.py
