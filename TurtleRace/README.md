## Overview

Interactive desktop game where you place a bet on a turtle color and watch six turtles race across the screen. Simple, immediate, and suitable as a small Python GUI exercise or demo.

### Features

Place a color bet via a prompt.
Six colored turtles race with randomized movement.
Winner is announced in the console.
Click the window to exit.

### Requirements

Python 3.x (no extra packages; uses the standard turtle module).

### Run
```python

python main.py

```

### How to play

- Run the program.

- When prompted, type one of the available colors (see list below) and submit.

- Watch the race. The winning color is printed to the console.

- Click the window to close it.
